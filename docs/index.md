# CKTT AI Agent Platform

Enterprise-grade AI Agent platform for building, deploying, and managing intelligent agents.

## Features

- **Multi-Agent Support**: ReAct, Conversational, and custom agent types
- **Tool System**: Extensible tool framework for agent capabilities
- **Memory Management**: Buffer and vector-based memory systems
- **Task Scheduling**: Async task processing with Celery
- **REST API**: FastAPI-based RESTful API
- **Docker Support**: Containerized deployment
